## Agent skills

### Issue tracker

Задачи хранятся локально в Markdown-файлах; внешние PR не являются источником задач. См. `docs/agents/issue-tracker.md`.

### Triage labels

Используются стандартные состояния: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. См. `docs/agents/triage-labels.md`.

### Domain docs

Проект использует single-context структуру. См. `docs/agents/domain.md`.
