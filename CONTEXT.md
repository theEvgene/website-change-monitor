# Мониторинг веб-страниц

Контекст персонального приложения, которое отслеживает выбранные части веб-страниц и сохраняет историю наблюдений.

## Language

**Монитор**:
Сохраненная настройка наблюдения за одной конкретной веб-страницей, включая Целевые селекторы, Селекторы исключения, периодичность и метки.
_Avoid_: Задача, сайт, подписка

**Приостановленный монитор**:
Монитор, исключенный из автоматических Проверок без удаления его истории и Базового снимка. Для него остается доступна ручная Проверка.
_Avoid_: Отключенный, удаленный, неактивный

**Проверка**:
Одна попытка исследовать страницу в рамках Монитора, результат которой сохраняется в его журнале.
_Avoid_: Запуск, сканирование, опрос

**Ручная проверка**:
Проверка, которую пользователь запускает командой «Запустить сейчас», не дожидаясь наступления срока.
_Avoid_: Тестовая проверка, принудительный запуск

**Просроченная проверка**:
Проверка, срок которой наступил, пока локальное приложение не работало; она должна начаться после следующего запуска приложения.
_Avoid_: Пропущенная проверка

**Повторная проверка**:
Проверка, автоматически назначаемая со сроком через одну минуту после ошибочного Результата проверки. У каждой исходной Проверки может быть не более одной Повторной проверки; для Приостановленного монитора она ожидает возобновления автоматических Проверок.
_Avoid_: Ретрай, повторная попытка, перезапуск

**Подготовка страницы**:
Ожидание отрисовки и однократная прокрутка до целевой области перед ее исследованием. В MVP не включает нажатия, переходы по пагинации или прокрутку бесконечной ленты.
_Avoid_: Навигация, сценарий взаимодействия

**Целевой селектор**:
CSS-селектор из обязательного непустого набора Монитора, указывающего значимые элементы страницы. Порядок ввода Целевых селекторов не задаёт порядок Снимка.
_Avoid_: Основной селектор, селектор области

**Целевая область**:
Упорядоченное по DOM множество уникальных элементов страницы, совпавших хотя бы с одним Целевым селектором. Элемент, совпавший с несколькими Целевыми селекторами, входит в него один раз.
_Avoid_: Единственный корневой элемент, кусок верстки, блок страницы

**Селектор исключения**:
CSS-селектор из необязательного набора Монитора, указывающего незначимые поддеревья внутри Целевой области. Порядок ввода Селекторов исключения не влияет на результат.
_Avoid_: Отклоняющий селектор, фильтр

**Исключение**:
DOM-поддерево внутри элементов Целевой области, совпавшее с Селектором исключения и полностью удаляемое из наблюдения вместе с текстом и вложенными элементами.
_Avoid_: Игнорируемый тег, скрытие

**Снимок**:
Нормализованное представление структуры и видимого текста Целевой области после применения Исключений. CSS-классы, атрибуты и стили помогают найти элементы, но сами в Снимок не входят.
_Avoid_: HTML, скриншот, версия страницы

**Область наблюдения**:
Совокупность URL, набора Целевых селекторов и набора Селекторов исключения Монитора. Изменение состава или значения любого из этих компонентов после подтверждения удаляет прежнюю историю Монитора; простая перестановка селекторов не меняет Область наблюдения.
_Avoid_: Суть содержимого, конфигурация Монитора

**История Монитора**:
Упорядоченный журнал Проверок и сохранённые Снимки текущей Области наблюдения. Полностью удаляется при подтверждённом изменении Области наблюдения или удалении Монитора.
_Avoid_: Архив сайта, журнал изменений

**Базовый снимок**:
Первый успешный Снимок новой Области наблюдения, с которым сравнивается следующая успешная Проверка. Его создание не считается Изменением и не вызывает уведомление.
_Avoid_: Старая база, исходный DOM

**Изменение**:
Различие между двумя Снимками: появление, исчезновение или перестановка значимых элементов либо изменение их видимого текста.
_Avoid_: Обновление сайта, изменение HTML

**Сравнение**:
Построчное представление двух последовательных сохраненных Снимков: удаленное выделено красным в прежнем состоянии, добавленное — зеленым в новом.
_Avoid_: Сравнение DOM, журнал изменений

**Результат проверки**:
Итог Проверки: без изменений, обнаружено Изменение либо ошибка. Если ни один Целевой селектор не нашёл ни одного элемента, Проверка завершается ошибкой и последний успешный Снимок остается базой для будущего сравнения.
_Avoid_: Статус Монитора

**Окончательная ошибка**:
Ошибочный Результат Повторной проверки, после которого цепочка Проверок завершена и создаётся Уведомление. Ошибка исходной Проверки до выполнения назначенной Повторной проверки окончательной не считается.
_Avoid_: Фатальная ошибка, первая ошибка

**Уведомление**:
Пользовательское сообщение, созданное из зафиксированного Результата проверки и направляемое в один или несколько каналов: центр уведомлений, открытую или фоновую вкладку браузера и Telegram. Ошибка доставки не изменяет Результат проверки.
_Avoid_: Алёрт, push, статус доставки

**Состояние доставки Telegram**:
Отметка Уведомления о текущем результате его отправки в Telegram: отправляется, отправлено либо не отправлено. Неотправленное Уведомление не доставляется задним числом после восстановления Telegram; новые Уведомления используют уже восстановленный канал.
_Avoid_: Статус проверки, очередь Telegram, повтор отправки

**Интервал проверки**:
Выбранный для Монитора период между завершением одной Проверки и сроком следующей: 6, 12, 24, 48 или 72 часа. После простоя выполняется только одна Просроченная проверка, без воспроизведения пропущенных запусков.
_Avoid_: Расписание, cron, частота обновления

**Метка**:
Необязательное именованное обозначение, которое может быть назначено нескольким Мониторам; у одного Монитора может быть несколько Меток. Фильтр по Метке показывает только назначенные ей Мониторы.
_Avoid_: Группа, категория, папка

**Режим контрольных уведомлений**:
Глобальный режим, при котором успешная Проверка без Изменений отправляет Уведомления в Telegram и через открытую или фоновую вкладку браузера, но не добавляет их в центр уведомлений. Действует до ручного выключения; Журнал Проверок ведется независимо от этого режима.
_Avoid_: Уведомлять всегда, heartbeat отдельного Монитора
