# Triage Labels

| Canonical role | Label in this tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Требует оценки сопровождающего |
| `needs-info` | `needs-info` | Ожидает информации от автора |
| `ready-for-agent` | `ready-for-agent` | Полностью определена и готова для AFK-агента |
| `ready-for-human` | `ready-for-human` | Требует участия человека |
| `wontfix` | `wontfix` | Не будет реализована |

Когда скилл упоминает каноническую роль, нужно использовать соответствующую строку из второго столбца.
